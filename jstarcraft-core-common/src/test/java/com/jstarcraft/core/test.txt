安装H2数据库
http://www.h2database.com/html/download.html
构建H2数据库
java -classpath h2*.jar org.h2.tools.Script -user sa -password "" -url jdbc:h2:~/database
启动H2数据库
java -classpath h2*.jar org.h2.tools.Server -tcp -tcpPort 9092 -tcpAllowOthers -web -webPort 8082 -webAllowOthers
